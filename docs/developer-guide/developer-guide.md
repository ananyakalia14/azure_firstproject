# Developer Guide

## Local Development

- Install frontend dependencies with `npm install` in the frontend folder.
- Install backend dependencies with `npm install` in the backend folder.
- Run the frontend locally with `npm run dev`.
- Use Azure Functions Core Tools for local API development when needed.

## Code Standards

- Follow strict TypeScript conventions.
- Keep UI components reusable and focused.
- Prefer server-side rendering where appropriate.
- Keep secrets in environment variables.
- Validate input and handle errors gracefully.

## Suggested Next Steps

- Implement Azure Blob upload integration.
- Connect Azure OpenAI for content generation.
- Persist study sessions in Cosmos DB.
- Add authentication with Azure Entra ID.
